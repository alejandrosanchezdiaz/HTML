Frases HTML

EM: Se utiliza para dar énfasis a una parte del texto

STRONG: Se utiliza para resaltar alguna otra parte

Q: Se utiliza para agregar comillas

CODE: Se utiliza para poner el texto como si fuera codigo de algun lenguaje de programación

ABBR: Se utiliza para poner Abreviaciones

ACRONYM: Se utiliza para poner acrónimos

DFN: Se utiliza para poner definiciones

KBD: Significa texto que debe introducir el usuario

SAMP: Se utiliza para mostrar un ejemplo de una salida de un programa

VAR: Se utiliza para mostrar que el texto es una variable
